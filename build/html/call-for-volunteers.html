Call for Volunteers Resource¶

A Call for Volunteers Resource may be used published by a project or other organization looking to be matched with volunteers for as part of an OpenTechResponse volunteer event.

Example¶

{
    "@context": "https://opentechresponse/api/jsonld/call-for-volunteers",
    "slug": "ventilator-project-10",
    "email": "covid-19@ventilator-project-10.org",
    "href": "https://ventilator-project-10.org/covid-19",
    "event": "https://opentechresponse/api/events/covid-19",
    "display_name": "Ventilator Project 10",
    "description": "Ventilator Project 10 is looking for volunteers to aid it its effort to create a low-cost 3D printable ventilator.",
    "volunteer_requests": [
        {
            "slug": "industrial-design-engineer",
            "display_name": "Industrial Design Engineer",
            "description": "We're looking for an industrial design engineer to aid in drafting 3D models that can be printed using widely available materials and 3D printers.",
            "skills": [
                {
                    "skill": "freecad",
                    "proficiency": "expert"
                },
                {
                    "skill": "slic3r",
                    "proficiency": "proficient"
                }
            ]
        },
        {
            "slug": "scada-developer",
            "display_name": "SCADA Developer",
            "description": "We're looking for a SCADA developer to implement the control system for the ventilator.",
            "skills": [
                {
                    "skill": "SCADA",
                    "proficiency": "expert"
                }
            ]
        }
    ]
}
