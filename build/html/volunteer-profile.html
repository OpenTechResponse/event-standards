Volunteer Profile Resource¶

A Volunteer Profile Resource may be published by an individual who is willing to participate as a volunteer in one or more volunteer events to be matched with volunteer opportunities.

Example¶

{
    "@context": "https://opentechresponse/api/jsonld/volunteer-profile",
    "slug": "some-name",
    "email": "somename@example.com",
    "href": null,
    "github_profile": "https://github.com/example",
    "gitlab_profile": "https://gitlab.com/example",
    "linkedin_profile": "https://linkedin.com/in/somename",
    "volunteer_events": [
        "https://opentechresponse/api/events/covid-19"
    ],
    "skills": [
        {
            "skill": "python",
            "proficiency": "expert"
        },
        {
            "skill": "javascript",
            "proficiency": "proficient"
        }
    ]
}
